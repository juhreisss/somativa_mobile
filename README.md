# somativa_mobile
Portfólio Mobile em Flutter e Dart desenvolvido para o onboarding de Dev Júnior na Digital Horizon. O app traz minha biografia, lista de projetos e links de redes sociais. Possui também uma tela interativa que gera um perfil personalizado baseado na escolha de linguagem, nível de conhecimento e área dev do usuário.


## 🚀 Funcionalidades do Aplicativo

O aplicativo cumpre todos os requisitos mínimos solicitados pelo cliente e adiciona uma funcionalidade interativa exclusiva para melhorar a experiência do usuário (UX):

* Tela de Apresentação:** Biografia profissional detalhada, destacando soft e hard skills com um design limpo e moderno.
* Lista de Projetos:** Exibição organizada dos principais projetos desenvolvidos, demonstrando a aplicação prática do conhecimento técnico.
* Links para Redes Sociais:** Botões interativos que direcionam o usuário de forma nativa para plataformas profissionais (GitHub, LinkedIn, etc.).
* Tela Interativa (Gerador de Perfil Dev):** Uma funcionalidade dinâmica onde o usuário pode selecionar suas preferências (Linguagem Favorita, Nível de Conhecimento e Área de Atuação) para gerar um perfil personalizado na hora.
